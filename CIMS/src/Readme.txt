This is my CIMS project Crime Information Management System.

Which take the input from user and stores it into the data base and performs the tasks according to the user like insertion , Deletion , searching , fetching data from the data base.

Step 1. Select the task which you want to perform from the given options.(1,2,3,4,5)

1. Press 1-> to Insert the crime data.
2. Press 2-> to Search or Enter Criminal's details.
3. Press 3-> to Search for a crime(Case) report.
4. Press 4-> to View monthly report of crimes.
5. Press 5-> to Exit.

Step 2. Perform operations regarding to it.
Step 3. Move to the previous stage.
Step 4. Exit.
